Building a Classic Snake Game with JavaScript

by Gurdeep Kaur | November 04, 2024

Category: JavaScript Games

Dive into the world of game development by creating the classic Snake Game.

This project involves building a dynamic game using JavaScript. It covers essential game logic, canvas rendering, collision detection, and score tracking.

In this article, I will guide you through the process of creating a classic Snake Game using HTML, CSS, and JavaScript. This game is a fun project that will help you understand important programming concepts such as game logic, event handling, and dynamic rendering.

Step 1: Setting Up the HTML Structure

<canvas id="gameCanvas" width="500" height="500"></canvas>

Step 2: Styling the Game with CSS

#gameCanvas {
      display: block;
      margin: 50px auto;
      background-color: #000;
    }

Step 3: Adding Functionality with JavaScript

const canvas = document.getElementById("gameCanvas");
    const ctx = canvas.getContext("2d");
    let snake = [{x: 5, y: 5}];
    let direction = "RIGHT";
    let food = {x: 10, y: 10};
    
    function gameLoop() {
      moveSnake();
      detectCollision();
      updateCanvas();
      setTimeout(gameLoop, 100);
    }
    
    function moveSnake() {
      let head = {...snake[0]};
      if (direction === "RIGHT") head.x++;
      if (direction === "LEFT") head.x--;
      if (direction === "UP") head.y--;
      if (direction === "DOWN") head.y++;
      snake.unshift(head);
      snake.pop();
    }
    
    function updateCanvas() {
      ctx.clearRect(0, 0, canvas.width, canvas.height);
      ctx.fillStyle = "green";
      for (let segment of snake) {
        ctx.fillRect(segment.x * 10, segment.y * 10, 10, 10);
      }
      ctx.fillStyle = "red";
      ctx.fillRect(food.x * 10, food.y * 10, 10, 10);
    }

Conclusion

This tutorial demonstrated how to build the classic Snake Game from scratch. With JavaScript, I was able to create a simple yet functional game that you can extend with additional features, such as levels, speed increases, and more complex game mechanics.

I hope you enjoyed building the Snake Game! Feel free to experiment with the design and features to create your own variations.
