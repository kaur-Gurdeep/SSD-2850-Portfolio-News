Creating a Simple Calculator with HTML, CSS, and JavaScript

by Gurdeep Kaur | October 15, 2024

Category: JavaScript Applications

Dive into the world of game development by recreating the classic Snake Game.

This project includes features like keyboard controls, collision detection, and score tracking, built using HTML Canvas and JavaScript. It demonstrates core game logic and canvas rendering techniques.

In this article, I will walk you through the process of creating a simple yet functional calculator using HTML, CSS, and JavaScript.

Step 1: Setting Up the HTML Structure

<div id="calculator">
          <input id="display" readonly>
          <div id="keys">
              <button class="operator" onclick="Display('+')">+</button>
              <button onclick="Display('7')">7</button>
              <button onclick="Display('8')">8</button>
              <button onclick="Display('9')">9</button>
              <button class="operator" onclick="Display('-')">-</button>
              <button onclick="Display('4')">4</button>
              <button onclick="Display('5')">5</button>
              <button onclick="Display('6')">6</button>
              <button class="operator" onclick="Display('*')">*</button>
              <button onclick="Display('1')">1</button>
              <button onclick="Display('2')">2</button>
              <button onclick="Display('3')">3</button>
              <button class="operator" onclick="Display('/')">/</button>
              <button onclick="Display('0')">0</button>
              <button onclick="Display('.')">.</button>
              <button onclick="calculate()">=</button>
              <button onclick="clearDisplay()">C</button>
          </div>
      </div>

Step 3: Adding Functionality with JavaScript

const display = document.getElementById("display");
      
      function Display(input) {
        display.value += input;
      }
      
      function clearDisplay() {
        display.value = "";
      }
      
      function calculate() {
        try {
          display.value = eval(display.value);
        } catch (error) {
          display.value = "Error";
        }
      }

Conclusion

In this project, I explored the fundamentals of web development by creating a simple yet functional calculator. I structured the layout using HTML, styled it with CSS, and added interactivity through JavaScript. This hands-on experience helped me enhance my understanding of essential web technologies and problem-solving skills. Working on this project was a great way to apply my knowledge and improve my confidence in building interactive web applications. I'm excited to continue learning and tackling more challenging projects in the future.
